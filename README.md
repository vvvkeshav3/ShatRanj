# ShatRanj

#### ShatRanj is a realtime multiplayer chess game where user can also video call his/her opponent friend.

#### This is the Frontend [React] part of the game.

### Tech Stack :
#### -Reactjs
#### -Nodejs and Expressjs
#### -Socket.io and simple-peer
#### -Chessjs
